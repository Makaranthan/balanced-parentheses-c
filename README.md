# Balanced Parentheses Checker (C)

## Description
Checks if an expression has balanced parentheses using stack.

## Features
- Supports (), {}, []
- Uses stack push/pop operations
- Menu-independent, simple CLI input

## How to Run
gcc balanced-parentheses.c -o checker
./checker
